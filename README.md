# Landing-Page

Proyecto de **Landing Page** desarrollado con **HTML** y **CSS**, enfocado en practicar estructura de sitio, estilos y maquetación.

## Descripción
En este repositorio construí una landing page estática utilizando:
- **HTML** para la estructura del contenido.
- **CSS** para el diseño visual (estilos, colores, tipografía y layout).

## Tecnologías usadas
- HTML5
- CSS3

## Qué incluye
- Estructura base de una página tipo landing.
- Secciones organizadas (contenido principal y bloques informativos).
- Estilos CSS para mejorar la presentación y la distribución de los elementos.

## Cómo ver el proyecto
1. Descarga o clona el repositorio.
2. Abre el archivo `index.html` en tu navegador.
